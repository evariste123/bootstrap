Variables & Data Types

Common Values

Python is dynamically typed. It infers the type based on the value assigned.

name = "EduHub"# String
age = 25# Integer
price = 19.99# Float
is_active = True# Boolean
nothing = None# NoneType

Data Structures

Lists & Dictionaries

# List (Ordered, Mutable)
fruits = ["apple", "banana"]
fruits.append("cherry")

# Dictionary (Key-Value Pairs)
user = {
    "id": 1,
    "name": "Pythonista"
}
print(user["name"])

Advanced Logic

List Comprehensions

A concise way to create lists in a single line.

nums = [1, 2, 3, 4, 5]

# Create a list of squares 
 for even numbers only
squares = [x**2 for x in 
 nums if x % 2 == 0]
# Output: [4, 16]

Conditionals

If / Elif / Else

score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")

Error Handling

Try / Except

Prevent your program from crashing when an error occurs.

try:
result = 10 / 0
except ZeroDivisionError:
print("Cannot divide by 
 zero!")
finally:
print("Execution complete.")

Object Oriented Programming

Classes & Inheritance

class Animal:
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return "Woof!"

my_dog = Dog()
print(my_dog.speak())

Reading & Writing
