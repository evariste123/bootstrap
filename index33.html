Detailed Q&A

Q: What is a Constructor?

A: A special method that is automatically called when an object of a class is created. It has the same name as the class.

Q: Difference between Array and Vector?

A: Arrays have a fixed size. Vectors (from <vector>) are dynamic and can resize automatically.

Q: What is the purpose of 'virtual' functions?

A: They are used in base classes to ensure that the correct function is called for an object, regardless of the expression used to make the function call (Polymorphism).

Q: What is 'Destructor'?

A: A destructor is a member function which is invoked automatically when the object goes out of scope or is explicitly destroyed by a call to delete.

Q: What is the 'this' pointer?

A: It is a constant pointer that holds the memory address of the current object. It is available within all non-static class member functions.
