    Loops & Iteration

    For & For...of

     
    // Standard For Loop    
for (let i = 0; i < 5; i++) {
    console.log(i);
}

 
    // Modern For...of 
    (Best for Arrays)    
const items = ["Apple", 
    "Orange"];
for (const item 
    of items) {
    console.log(item);
}

    Array Power Methods

    Map, Filter, Reduce

    const nums = [1, 2, 3, 4];

const doubled = nums.map 
    (n => n * 2);     
    // [2, 4, 6, 8]
const evens = nums.filter 
    (n => n % 2 === 
    0); // [2, 4]
const sum = nums.reduce 
    ((acc, cur) => acc + cur, 0); 
    // 10
