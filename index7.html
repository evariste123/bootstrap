Documentation & endpoints

Base URL: https://api.ourapi.example/v1

GET /data — Fetch data. Query params: limit, cursor. POST /items — Create item. Send JSON body. GET /status — Service health and quota usage.

Authentication

Send your API key in the Authorization header: Authorization: Bearer <API_KEY>. Do not share keys publicly.

Rate limits

Trial accounts: 100K requests per month, burst-friendly but subject to fair use. Production plans offer higher limits and SLA.
