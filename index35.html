Connect XAMPP to CMD

Step-by-Step Setup

  • Open XAMPP Control Panel and start MySQL.
  • Open Command Prompt (CMD).
  • Navigate to the MySQL bin directory:
cd C:\xampp\mysql\bin
  • Log in to the MySQL server (default user is 'root' with no password):
mysql -u root -p

Press Enter when asked for a password (leave it blank).

Database Management

Commands

CREATE DATABASE eduhub_db;  -- Create a new DB
SHOW DATABASES;             -- List all DBs
USE eduhub_db;              -- Enter the DB
DROP DATABASE eduhub_db;    -- Delete DB

Table Properties & Creation

Table Setup with Constraints

CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    age INT DEFAULT 18,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Querying Data

Filtering and Sorting

-- Select all users over 20 sorted by name
SELECT * FROM users 
WHERE age > 20 
ORDER BY username ASC;

-- Pattern matching
SELECT * FROM users WHERE email LIKE '%@gmail.com';

Table Relations (Joins)

Inner Join Example

SELECT orders.id, users.username
FROM orders
INNER JOIN users ON orders.user_id = users.id;

Advanced Functions

Math & Grouping

SELECT COUNT(id) AS total_users FROM users;
SELECT AVG(age) FROM users;
SELECT SUM(price) FROM orders GROUP BY status;
