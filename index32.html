Primitives & Variables

Data Types

Java is statically typed. Every variable must have a declared type.

int myNum = 5; 
// Integer
float myFloatNum = 5.99f; 
// Floating point
char myLetter = 'D'; 
// Character
boolean myBool = true;
// Boolean
String myText = "Hello";
// String (Object)
