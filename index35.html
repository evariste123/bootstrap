Connect XAMPP to CMD

Step-by-Step Setup

  • Open XAMPP Control Panel and start MySQL.
  • Open Command Prompt (CMD).
  • Navigate to the MySQL bin directory:
cd C:\xampp\mysql\bin
  • Log in to the MySQL server (default user is 'root' with no password):
mysql -u root -p

Press Enter when asked for a password (leave it blank).

Querying Data

Filtering and Sorting

-- Select all users 
 over 20 sorted by name
SELECT * FROM users 
WHERE age > 20 
ORDER BY username ASC;

-- Pattern matching
SELECT * FROM users WHERE 
 email LIKE '%@gmail.com';

Table Relations (Joins)

Inner Join Example

SELECT 
 orders.id, users.username
FROM orders
INNER JOIN 
 users ON
 orders.user_id = users.id;
