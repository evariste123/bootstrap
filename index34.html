    Form Handling

    $_POST & $_GET

    Superglobals are built-in variables that are always available in all scopes.

    // Handling a POST form
if ($_SERVER["REQUEST_METHOD"
    ] == "POST") {
$user = $_POST['username'];
echo "Welcome, " . $user;
}
