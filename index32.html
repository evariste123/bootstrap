Primitives & Variables

Data Types

Java is statically typed. Every variable must have a declared type.

int myNum = 5;               // Integer
float myFloatNum = 5.99f;    // Floating point
char myLetter = 'D';         // Character
boolean myBool = true;       // Boolean
String myText = "Hello";     // String (Object)

Strings & Casting

String Methods

String txt = "EduHub Java";
System.out.println(txt.length());     // 11
System.out.println(txt.toUpperCase());// EDUHUB JAVA
System.out.println(txt.indexOf("Java")); // 7

Loops & Logic

For-Each Loop

Commonly used to iterate through arrays or collections.

String[] cars = {"Volvo", "BMW", "Ford"};
for (String i : cars) {
  System.out.println(i);
}

Collections

ArrayList (Dynamic)

import java.util.ArrayList;

ArrayList<String> list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.remove(0);

OOP Pillars

Constructors & Methods

public class User {
  String name;

  // Constructor
  public User(String n) {
    name = n;
  }

  public void sayHello() {
    System.out.println("Hi " + name);
  }
}

Interfaces

Abstraction

interface Animal {
  public void animalSound(); // interface method
}

class Pig implements Animal {
  public void animalSound() {
    System.out.println("Oink Oink");
  }
}

Lambda Expressions

Functional Programming

Added in Java 8 to simplify code syntax.

ArrayList<Integer> numbers = new ArrayList<>();
numbers.add(5);
numbers.add(9);
numbers.forEach( (n) -> { System.out.println(n); } );

Exception Handling

Try...Catch

try {
  int[] myNumbers = {1, 2, 3};
  System.out.println(myNumbers[10]);
} catch (Exception e) {
  System.out.println("Error: Index out of bounds.");
} finally {
  System.out.println("Cleanup code here.");
}
